# Aurum — Luxury Cafe Website

A cinematic, luxury cafe website built with **Next.js 14**, **Framer Motion**, and **Tailwind CSS**.

## Design Philosophy

Deep espresso tones, liquid gold accents, and generous negative space create an atmosphere of restrained opulence. Cormorant Garamond typography lends editorial gravitas while Framer Motion orchestrates silky scroll-driven reveals.

## Features

- **Hero** — Full-bleed video with parallax + scroll fade
- **Story** — Cinematic image reveal with clip-path animation
- **Menu** — Filterable grid with hover microinteractions
- **Video Interlude** — Scroll-scale cinematic break
- **Atelier** — Parallax experience rows with scroll-driven motion
- **Gallery** — Masonry image grid with gold corner accents
- **Reservations** — Elegant form with validation
- **Custom Cursor** — Magnetic gold dot + follower ring
- **Navigation** — Transparent → frosted glass on scroll

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- **Next.js 14** (App Router)
- **Framer Motion 11** — scroll-driven animations, clip-path reveals, parallax
- **Tailwind CSS 3** — utility-first styling
- **Google Fonts** — Cormorant Garamond, Jost, Playfair Display
- **Unsplash** — high-quality photography
- **Pexels** — cinematic video backgrounds

## Current Development

Aurum is evolving beyond a showcase website into a complete digital ecosystem for modern cafés.

I'm currently building a fully integrated **Point of Sale (POS)** and **Kitchen Display System (KDS)** designed specifically for the Aurum experience.

### Planned Features

- **Real-time order management** using Socket.IO
- **Interactive POS** for dine-in, takeaway, and delivery orders
- **Kitchen Display System (KDS)** with live ticket updates and preparation workflow
- **Order status tracking** from placement to completion
- **Table management** and reservation synchronization
- **Inventory-aware menu** with item availability updates
- **Role-based dashboards** for cashiers, kitchen staff, and managers
- **Sales analytics** and business insights
- **Multi-device synchronization** across tablets, desktops, and kitchen displays

The goal is to transform Aurum from a luxury café website into a production-ready hospitality platform that combines elegant customer experiences with efficient restaurant operations.
